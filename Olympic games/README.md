# Bienvenidos al Análisis del DataFrame Juegos Olímpicos

Análisis de datos de atletas olímpicos usando Python, Pandas y Matplotlib.

## Descripción

Este script carga un dataset de *atletas olímpicos* y lo limpia para facilitar su análisis.  
Después, permite hacer filtros por país, apellido, altura y peso, además de *generar gráficos para visualizar la distribución de los datos*.

## Funcionalidades

- Carga de datos desde un archivo Excel.
- Renombrado de columnas para hacerlas más legibles.
- Unión con un archivo CSV de regiones NOC.
- Sustitución de valores nulos por `"Sin Datos"` en columnas de texto.
- Conversión de altura y peso a valores numéricos.
- Filtrado de atletas por país, apellido y condiciones específicas.
- Ordenación por altura.
- Gráfico de distribución de altura.
- Gráfico de atletas por país.
- Filtro de atletas de Madrid con menos de 70 kg.

# Welcome to the Olympic Games DataFrame Analysis

Analysis of Olympic athlete data using Python, Pandas, and Matplotlib.

## Description

This script loads a dataset of *Olympic athletes* and cleans it to facilitate analysis.

It then allows filtering by country, last name, height, and weight, as well as *generating graphs to visualize the data distribution*.

## Features

- Loading data from an Excel file.
- Renaming columns for better readability.
- Joining with a CSV file of NOC regions.
- Replacing null values ​​with "No Data" in text columns.
- Converting height and weight to numeric values.
- Filtering athletes by country, last name, and specific conditions.
- Sorting by height.
- Graphing height distribution.
- Graphing athletes by country.
- Filtering for athletes from Madrid weighing less than 70 kg.
